**MediaShrink** — это универсальный инструмент для сжатия медиа-контента. Приложение работает только локально, не содержит рекламы и не отслеживает ваши действия.

---

## 🎵 Аудио
* **Форматы (контейнеры):** MP3, M4A, WAV, OGG, FLAC, AMR.
* **Кодеки обработки:** AAC (основной кодек для сжатия и вывода в `.m4a`).

## 🖼 Фото и GIF
> **В разработке.** Функции сжатия изображений и анимированных форматов находятся в стадии активной разработки и будут добавлены в будущих обновлениях.
