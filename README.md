## ⚡ Философия проекта
Большинство современных конвертеров на GitHub тащат за собой огромные бинарники и тяжелые UI-фреймворки. MediaShrink создан с другой целью:
1. **Нулевой визуальный мусор:** Абсолютно черный AMOLED-интерфейс (`#000000`). Никаких лишних тем оформления — экономит заряд батареи и оперативную память.
2. **Экстремальная легкость:** APK-файл весит менее 3 МБ.
3. **Оптимизация ОЗУ:** Стабильно работает и не вылетает даже в условиях, когда системе доступно менее 300 МБ свободной оперативной памяти. 

## 🛠 Технический стек
Приложение построено на современных и чистых технологиях:
* **Язык:** Kotlin
* **Движок:** `androidx.media3:media3-transformer` 
* **Среда разработки:** Проект написан и скомпилирован полностью на мобильном устройстве (AndroidIDE / Android Code Studio).

## 🎛 Возможности (v0.1-alpha)

### Аудио-движок
Продвинутый контроль над аудио-дорожками:
* **Hi-Fi (256 kbps, AAC)** — максимальное качество.
* **Standard (128 kbps, AAC)** — идеальный баланс для мессенджеров.
* **Voice / Lecture (64 kbps, Mono)** — умный даунмикс каналов в моно-режим, позволяющий сэкономить до 50% веса файла при сжатии голосовых записей.

### Видео-движок (В разработке)
* Аппаратное кодирование H.264/H.265 (HEVC).
* Сохранение оригинального разрешения с умным понижением битрейта.
