## ✨ Почему стоит выбрать MediaShrink?

Большинство приложений для сжатия медиа либо напичканы рекламой,
либо отправляют ваши файлы на сторонние серверы. **MediaShrink работает иначе:**

- 🔒 Ваши файлы никогда не покидают устройство
- 🚫 Ни одной рекламы, ни единой подписки
- 📴 Работает даже без интернета
- ⚡ Минимум лишнего — только нужный функционал
- 🆓 Открытый код — проверяйте сами
