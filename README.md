**MediaShrink** — это универсальный инструмент для сжатия медиа-контента. Приложение работает только локально, не содержит рекламы и не отслеживает ваши действия.

## 🎵 Аудио
* **Поддерживаемые форматы:** MP3, AAC, M4A, FLAC, WAV, OGG.

## 🖼 Фото
> **В разработке.** Функции сжатия изображений находятся в стадии активной разработки и будут добавлены в будущих обновлениях.

## 📹 Видео
> **В разработке.** Функции обработки видео находятся в стадии активной разработки и будут добавлены в будущих обновлениях.
