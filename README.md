**MediaShrink** — легковесный и приватный инструмент для сжатия медиа-контента на Android. Никакой рекламы, никакого трекинга, никакого интернета — вся обработка происходит локально на вашем устройстве.

## ✨ Почему MediaShrink?

- 🔒 **100% приватность** — файлы никогда не покидают устройство
- 🚫 **Без рекламы** и подписок
- 📴 **Работает офлайн** — интернет не требуется
- ⚡ **Быстро и легко** — минималистичный интерфейс без лишних функций
- 🆓 **Open Source** — полностью открытый исходный код
